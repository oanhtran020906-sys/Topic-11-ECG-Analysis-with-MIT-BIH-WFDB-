# Time Series Analysis and Forecasting — Course Site

Static site for EP16.TOKT11122 hosted with GitHub Pages.

## View
- GitHub Pages: `https://nghianguyen7171.github.io/ts_course/`

## Develop
1. Open the repo root.
2. Serve locally (any static server), e.g.:
   - Python: `python -m http.server 8000`
3. Visit `http://localhost:8000`.

## Structure
- `index.html` — main page using content from `Syllabus_alt.md` (includes light/dark toggle).
- `style.css` — layout and theme variables for light/dark modes.
- `img/` — logos, instructor photo, textbook covers.
